# 🌤 LuckyForecast

LuckyForecast is a responsive weather forecasting web application built using HTML, CSS, JavaScript, and the OpenWeather API. It provides real-time weather information, location-based weather updates, hourly forecasts, sunrise and sunset timings, and dynamic weather-based themes.

## 🚀 Live Demo

https://luckyfarog065-netizen.github.io/LuckyForecast/

## ✨ Features

* 🌍 Search weather for any city worldwide
* 📍 Automatic location-based weather detection
* 🌡 Real-time temperature updates
* 💧 Humidity information
* 💨 Wind speed details
* 🌅 Sunrise timings
* 🌇 Sunset timings
* 📅 Hourly weather forecast
* 🌙 Day and Night theme support
* 🎨 Dynamic weather-based backgrounds
* 📱 Responsive design for desktop and mobile devices
* ⚡ Fast and lightweight user interface

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeather API
* GitHub Pages
